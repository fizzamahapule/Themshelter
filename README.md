# 🏠 TempShelter Project

A Python Flask-based web application for disaster housing allocation system.  
It helps in assigning shelters to affected people based on priority.

---

## 🚀 Features
- Priority-based allocation system  
- Easy-to-use interface  
- Fast decision making during disasters  
- Simple and lightweight web app  

---

## 🛠️ Technologies Used
- Python  
- Flask  
- HTML / CSS (if used)  

---

## 📦 Installation

### 1. Clone the repository
git clone https://github.com/your-username/tempshelter.git

### 2. Move into project folder
cd tempshelter

### 3. Install dependencies
pip install -r requirements.txt

---

## ▶️ How to Run

python app.py

Then open in browser:
http://127.0.0.1:5000/
